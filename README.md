## Getting Started

First, set up the json-server:

```bash

npx json-server --watch db.json --port 3001

```

To run it in development mode:

```bash
npm run dev
# or
yarn dev
```
To build it you have to first run the json-server and then 

```bash
npm run build
```

when the build it's finished run: 

```bash
npm start
```
